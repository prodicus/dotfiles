## dotfiles

@tasdikrahman does **dotfiles**

I am using [rcm](https://github.com/thoughtbot/rcm) and [git](https://git-scm.com) to manage  my `~/.dotfiles` and keep them in sync.

For updating single file using `rcup`, you can do a

```sh
$ rcup <filename>
```
